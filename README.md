# Šabanovići — Porodično stablo

A family tree website for the Šabanović family.

## Prerequisites

- Python 3 (pre-installed on most macOS and Linux systems)

## How to start

1. Open a terminal and navigate to the project folder:
   ```bash
   cd /path/to/Family-tree
   ```

2. Start a local web server:
   ```bash
   python3 -m http.server 8000
   ```

3. Open your browser and go to:
   ```
   http://localhost:8000
   ```

## Usage

- **Stablo** — view the family tree. Drag to pan, scroll to zoom, or use the +/- buttons.
- **Zanimljivosti** — view fun facts about the family.

## Updating family data

Edit `data/family.json` to add or modify family members. Edit `data/funfacts.json` to update fun facts. Refresh the browser to see changes.
